# 购买记录（网页版）

你之前提的需求是“**在网页中使用**”，所以这个版本改成了浏览器可直接打开的网页工具。

## 功能

- 记录物品名称、购买日期、价格
- 自动计算使用至今的天数（按自然日，至少 1 天）
- 自动计算每日成本 = 价格 / 使用天数
- 本地持久化（使用浏览器 `localStorage`）
- 支持单条删除和一键清空

## 如何使用

直接双击打开 `index.html`，或用本地静态服务打开：

```bash
python3 -m http.server 8000
```

然后访问：<http://localhost:8000>

## 数据说明

- 数据保存在当前浏览器的 `localStorage`（键名：`purchaseRecords`）
- 不会自动同步到服务器
- 清除浏览器站点数据会导致记录丢失

## 审查结论（针对旧版 CLI）

旧版是命令行脚本，不适合直接嵌入网页交互；本次改为前端页面后，才能满足“网页中使用”的场景。
