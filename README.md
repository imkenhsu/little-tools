# 购买记录（Vue 3 网页版）

已改为 **Vue 3 技术栈**，当前界面风格更接近轻量数据库面板，适合日常记录和快速筛选。

## 功能

- 记录物品名称、购买日期、价格
- 自动计算使用至今天数（至少 1 天）
- 自动计算每日成本 = 价格 / 使用天数
- 展示总支出
- `localStorage` 本地持久化
- 自动过滤本地存储中的无效旧数据
- 支持按物品名搜索、按日期/价格排序
- 支持按分类筛选与分类统计
- 支持单条记录页内编辑
- 支持导出 JSON、展开导入面板后粘贴导入或文件导入并合并历史记录
- 单条删除、清空全部

## 运行方式

直接打开 `index.html`，或用静态服务：

```bash
python3 -m http.server 8000
```

访问：<http://localhost:8000>

## 技术说明

- Vue 3 通过 ESM CDN 引入（`unpkg`）
- 首次运行需要能访问 CDN，并非完全离线
- 无需打包工具即可运行
- 业务逻辑已拆到 `purchase-utils.js`
- 数据键名：`purchaseRecords.v2`
